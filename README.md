# paramx-templetes

- this repo is for prewritten templates for paramx tool from cyinnove

## Templates
-the syntax is basic
```
tag: {{TAG_NAME}} // xss,sqli,ssrf, as you want you can create your own
part: {{PART_NAME}} // query,subdomain, ... will add new parts in z future

list:
  - param1
  - param2
  - param2
```
